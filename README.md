**Road Accident Analysis Dashboard**

Project Overview

Road accidents are a major public safety concern that result in significant loss of life and injuries every year. Understanding the patterns and factors that contribute to these accidents is important for improving road safety.
This project analyzes road accident data and presents insights through an interactive dashboard built using Microsoft Power BI Desktop. The dashboard helps identify accident trends, severity distribution, and contributing factors such as weather conditions, road types, and vehicle involvement.

Problem Statement
Road accidents occur due to various factors such as weather conditions, road infrastructure, and vehicle involvement. Large volumes of accident data are collected, but identifying meaningful patterns and understanding the risk factors behind accidents can be challenging. Analyzing this data helps reveal trends and contributing factors affecting road safety.


Proposed Solution
The solution involves analyzing road accident data and presenting the insights using an interactive Power BI dashboard.

Steps followed in the project:
1. Collect road accident dataset.
2. Clean and prepare the dataset for analysis.
3. Import the dataset into Power BI.
4. Create calculated measures such as accident count, total fatalities, and total casualties.
5. Develop visualizations to analyze accident patterns.
6. Add filters and slicers for interactive data exploration.
7. Generate insights about accident trends and risk factors.

Dashboard Features
The dashboard provides several insights including:
* Total accidents, fatalities, and casualties (KPI cards)
* Accident severity distribution
* Weather conditions vs accident count
* Vehicles involved vs casualties relationship
* Accident trends over time
* Accidents by road type
* Vehicle type distribution
* Interactive filters for weather conditions

Key Insights
* Weather conditions significantly influence accident occurrences.
* Some road types show higher accident frequency.
* Accidents involving more vehicles tend to result in more casualties.
* Accident trends vary over different years.

Conclusion
This project demonstrates how data visualization can help analyze road accident data and identify patterns and risk factors. The insights obtained from the dashboard can support better understanding of accident conditions and help improve road safety awareness.
